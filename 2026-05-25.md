### Redis 数据结构与缓存实战

📚 [中间件] Redis 数据结构与缓存实战

>> 核心要点

1. 要点一：五种核心数据结构——String（计数器/分布式锁）、Hash（对象缓存）、List（消息队列）、Set（去重/标签）、Sorted Set（排行榜/延迟队列）
2. 要点二：缓存三大模式——Cache Aside（旁路缓存，最常用）、Read/Write Through（缓存代理）、Write Behind（异步写入）
3. 要点三：RTD 场景应用——用 Redis 缓存设备状态、Pub/Sub 实现机台告警推送、Sorted Set 做 Lot 优先级队列

>> 一句话总结
Redis 不是缓存数据库，而是数据结构服务器——用对数据结构，事半功倍。
